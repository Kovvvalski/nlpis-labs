How to Use the Word Table Editor

1. Upload a file: Use the "Choose File" button to upload a .txt or .rtf file that contains a list of words.

2. Filter Words: Use the filter section to filter words by their base word and part of speech.

3. Add, Edit, and Delete Words: Add new words by clicking "Add New Word". You can edit a word’s base or part of speech and delete a word.

4. Save and Load Data: Save your word list as a JSON file or load a previously saved JSON file.
